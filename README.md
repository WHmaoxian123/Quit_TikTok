# 🚫 Quit TikTok (短视频防沉迷锁)

> 斩断无意识的信息信息流摄入，夺回你对时间的绝对掌控权。

Quit TikTok 是一款极其硬核的 Android 端短视频防沉迷应用。它抛弃了温和的“提醒”模式，采用强有力的限制机制，帮助使用者戒断对短视频（如抖音、TikTok等）的依赖，将节省下来的精力投入到更有价值的深度学习与真实生活中。

## ✨ 核心机制 (Features)

*   **⏱️ 强制冷却周期**：采用 24 小时循环重置的锁定机制，一旦触发限制，在冷却期结束前绝不妥协。
*   **🔒 沉浸式拦截**：底层级别接管与监控，防止通过常规手段绕过防沉迷系统。
*   **⚡ 极简纯粹**：专注核心业务逻辑，无多余后台驻留，省电且轻量。

## 🎨 UI 与背景图自定义 (Important: Customization)

为了规避版权争议，本开源版本默认使用了纯黑色的占位背景图。

如果你想让界面更有沉浸感，可以将其替换为你喜欢的二次元角色壁纸、风景图或其他任意图片。**但请务必严格遵循以下替换规则，否则会导致项目编译失败：**

1. 准备好你的三张图片。
2. 找到项目目录中的图片文件夹：`app/src/main/res/drawable/`
3. 将你的图片**重命名为与占位图完全一致的文件名和后缀**，并替换掉原文件：
   *   `bg_1.png` (必须是 .png 格式)
   *   `bg_2.jpg` (必须是 .jpg 格式)
   *   `bg_3.png` (必须是 .png 格式)

> **⚠️ 注意**：请仅替换这三张图片，不要改动该目录下的 `bg_breath_circle.xml` 等代码文件。

## 🛠️ 本地编译与运行 (Getting Started)

1. 克隆本项目到本地：
   ```bash
   git clone [https://github.com/Lvasi/Quit_TikTok.git](https://github.com/Lvasi/Quit_TikTok.git)
   ```
2. 使用 Android Studio 打开本项目所在文件夹。
3. 等待 Gradle 自动同步依赖（首次可能需要几分钟）。
4. 连接你的 Android 手机（开启 USB 调试模式）或使用模拟器，点击 Run 按钮即可编译安装。
